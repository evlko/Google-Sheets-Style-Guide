Google Sheets Style Guide

Idea

This is a set of guidelines that can help separate entities in your tables. It can be useful to easily find the right ones and distinguish them from each other. Also, since named functions can be exported, naming them is especially important.

The guide will be expanded and updated from time to time, so feel free to share, comment, and suggest improvements.

Rules

1 Naming Rules

1.1 Named Ranges

Uppercase with underscores: NAMED_RANGE.

This naming allows you to distinguish your ranges from all functions and keeps the appearance of the letters in the global space ('cause all of them are uppercase).

1.2 Coefficients

The coefficient is a named range consisting of one cell and which depends on other cells. To such a range add _COEFF to the end: STEAM_REVENUE_COEFF.

1.3 Named Functions

Uppercase starting & with underscores: _NAMED_FUNCTION.

Named functions are worth starting with the underscore to filter them out from the standard ones. Standard functions use a dot as a separator but named functions can only have the underscore.

1.4 Function Arguments

Lowercase with underscores: arg_name.

You can use named ranges in functions, so it's better to use a different case to distinguish between them and func args.

1.5 Sheets

PascalCase but with blanks: Sheet Name.

2 Sheets

2.1 General

Main Sheet. Contains all the necessary information about the table (purpose, what you need to know, references, color rules). It always comes first.

2.2 Coefficients

Coefficients Sheet. Contains the calculation of all possible coefficients. More about them in 1.2.

2.3 Pivot

Pivot Table. Since such a table is created on the basis of some sheet and there may be several of them. Start with Pivot, then the name of the base sheet, then the number: Pivot Sheet Name 1
